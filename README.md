# image-to-code skill

Skill para Claude Code que convierte imágenes de diseño en HTML/CSS semántico, 
accesible y con buenas prácticas de SEO.

## Instalación
```bash
mkdir -p .claude/skills/image-to-code && curl -L "https://github.com/bgarcias/image-to-code/raw/main/image-to-code/SKILL.md" -o .claude/skills/image-to-code/SKILL.md
```
